How to Access the a321-wireless Entertainment System

Wählen Sie OZ WiFi auf einem Gerät, das sich mit dem WLAN-Netzwerk verbinden kann.

Öffnen Sie dann Chrome, Safari oder Firefox.

Sie werden automatisch zum Asiana-Portal weitergeleitet. Falls die Seite nicht geladen wird, geben Sie ozwifi.flyasiana.com ein.

Notice

  • Das Bordunterhaltungssystem ist für Chrome, Safari und Firefox optimiert.
  • Das Bordunterhaltungssystem funktioniert auf Geräten ab Android 5.0 und ab iOS 9.
  • Während der Nutzung dieser Dienste werden Kopfhörer empfohlen, um eine ruhige Umgebung im Flugzeug zu gewährleisten.
  • Das WLAN während des Fluges kann nur für den Zugriff auf den von Asiana Airlines angebotenen Unterhaltungsservice verwendet werden.
  • Stellen Sie während des Fluges alle elektronischen Geräte auf Flugmodus.
