무선 엔터테인먼트 시스템 접속 방법

How to Access the a321-wireless Entertainment System

와이파이 가능한 기기에서 ‘OZ WiFi’를 선택하십시오.

Select Oz WiFi on a device that can connect to the WiFi network.

Chrome, Safari, Firefox 중 하나의 웹 브라우저를 선택하여 실행하십시오.

Select Chrome, Safari, or Firefox, and open it.

아시아나 포털로 자동 연결됩니다.연결이 안될 경우 ozwifi.flyasiana.com을 입력하십시오.

You will be automatically directed to the Asiana portal. If the page does not load, enter ozwifi.flyasiana.com.

iOS(애플)이용자 - 탑승 전 미디어 플레이어 설치 (필수)

iOS(Apple) - Install the Media Play Before Boarding

1. 앱스토어에서 ‘Cabin Media Player’ 검색 후 무료로 다운로드 가능합니다.

2. 안드로이드나 노트북 이용자는 미디어 플레이어 설치가 필요하지 않습니다.


1. Download the Cabin Media Player for free from the App store.

2. The Cabin Media Player is not needed for Android or notebook users.

유의사항(Notice)

  • 서비스는 Chrome, Safari, Firefox에 최적화되어 있습니다.
  • 서비스는 안드로이드 기기 5버전 이상, iOS 기기 9버전 이상의 운영체제에서 작동합니다.
  • 서비스 이용 시 기내 정숙을 위해 이어폰 사용을 권장합니다.
  • 와이파이는 당사 서비스만 이용 가능합니다(인터넷, SNS 등 사용 불가).
  • 기내에서는 비행기 모드를 유지해주십시오.
  • The In-flight Entertainment System is optimized for Chrome, Safari, and Firefox.
  • The In-flight Entertainment System operates in devices with Android 5.0 or higher and iOS 9 or above versions.
  • Earphones are recommended while using these services to maintain a quiet environment in the plane.
  • The In-flight Wi-Fi can only be used to access the Wireless Entertainment Service offered by Asiana Airlines.
  • Set and keep all electronic devices on flight mode during the flght.
