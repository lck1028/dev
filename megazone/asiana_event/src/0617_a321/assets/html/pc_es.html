Cómo acceder al entretenimiento inalámbrico

Antes de embarcar
Traiga sus auriculares

Traiga sus auriculares para disfrutar de horas de entretenimiento.

Antes de embarcar
Instalación de ‘Cabin Media Player’ App - iOS (Apple)

1. Descárguese ‘Cabin Media Player’ gratis desde la App Store.

2. Los usuarios de Android o portátiles no necesitan ‘Cabin Media Player’.

How to Access the a321-wireless Entertainment System

Seleccione OZ WiFi en un dispositivo que pueda conectarse a la red wifi.

Seleccione Chrome, Safari o Firefox y ábralo.

Se conectará automáticamente con el portal de Asiana. Si la página no se carga, introduzca ozwifi.flyasiana.com.

Notice

  • El sistema de entretenimiento durante el vuelo está optimizado para Chrome, Safari y Firefox.
  • El sistema de entretenimiento durante el vuelo funciona en dispositivos con Android 5.0 e iOS 9 o versiones superiores.
  • Se recomienda utilizar auriculares mientras se utilizan estos servicios para mantener un entorno tranquilo en el avión.
  • El wifi del vuelo solo puede utilizarse para acceder al servicio de entretenimiento inalámbrico ofrecido por Asiana Airlines.
  • Configure y mantenga todos los dispositivos electrónicos en modo vuelo durante el vuelo.
